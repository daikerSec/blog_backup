PHP危险函数总结(随时补充)

By daiker

發表於 2017-07-09

1. 获取当前文件路径

  • echo dirname(__FILE__) ;
  • echogetcwd();

2. 获取同目录下的所有文件名称

  • echo var_dump(glob('./*'));

  • system("ls")/system("dir")/system-->可换成执行系统命令的函数;

    3. 读取某个文件的内容

  • echo file_get_contents("flag.php");

  • echo fgets(fopen("flag.php","r"));
  • show_source("flag.php");
  • echo fread(fopen("flag.php","r"),filesize("flag.php"));
  • echo var_dump(file("flag.php"));
  • copy("flag.php","daiker.txt");
  • include 'php://filter/read=convert.base64-encode/resource=flag.php';
  • highlight_file("flag.php");

4. 执行系统命令的函数

  • system
  • passthru
  • exec
  • pcntl_exec
  • shell_exec

5. 执行PHP的函数

  • eval(注意，不是函数，不能再回调函数里面使用)
  • assert
  • preg_replace
  • `

6. 回调函数

  • call_user_func
  • call_user_func_array
  • create_function
  • array_map
